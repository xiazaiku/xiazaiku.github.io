配置需求

为了确保流畅的游戏体验，请参考以下配置要求：

  • 最低配置：
    • 操作系统: Windows 10/Windows 11 (64位)
    • 处理器: Intel i5-4460 / AMD Ryzen 3 1200
    • 内存: 8 GB RAM
    • 显卡: NVIDIA GTX 770 / AMD Radeon RX 570
    • DirectX 版本: 11
    • 存储空间: 286 GB 可用空间
  • 推荐配置：
    • 操作系统: Windows 11 (64位)
    • 处理器: Intel Core i7 9800X / AMD Ryzen 7 Pro 2700X
    • 内存: 16 GB RAM
    • 显卡: NVIDIA RTX 2080 / AMD Radeon VII
    • DirectX 版本: 11
    • 存储空间: 286 GB 可用空间

重要提示：

  1. 请确保你的系统是Windows 10/Windows 11 (64位)完整版，不支持精简版系统。
  2. 单机模式下，游戏需要联网才能启动。
  3. 建议使用键鼠或飞行摇杆进行游戏，手柄可能会导致闪退。
  4. 进入游戏后，请选择离线模式游玩，在线模式可能会无限加载。
  5. 如果游戏语言变为英文，请在设置中重新选择简体中文。
  6. 如遇到ESC键退出无效的情况，请使用Alt+F4强制关闭，并在任务管理器中结束“FlightSimulator.exe”进程。
