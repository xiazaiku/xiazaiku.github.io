Android App Usage(应用监控) v5.75 高级版


App Usage 是一款监控安卓手机应用使用的强大工具。它能详细记录您每天使用各个应用的时长，帮助您掌握自己的数字生活，并进行有效的时间管理。类似于爱今天APP，App Usage专注于应用使用时间的精准记录。

App Usage 提供详尽的应用使用时长统计，协助您更好地管理应用使用时间，并提供便捷的应用批量卸载功能以及各种应用管理相关工具。

核心功能：

★ 应用使用历史： 追踪并显示您在每个应用上花费的时间，让您对自己的使用习惯一目了然。

★ 手机解锁历史： 记录您每天解锁手机的次数，帮助您了解自己对手机的依赖程度。

★ 活动历史： 详细记录您打开各个应用的时间，方便您回顾自己的使用轨迹。

★ 通知历史: 统计您收到的通知数量，并分析哪些应用发送的通知最为频繁，避免无用信息*扰。

★ 电池记录: 以图表形式展示电池使用情况，帮助您了解哪些应用耗电量大，并进行优化。

★ 过度使用提醒： 当您在手机或某个应用上花费过多时间时，及时发出提醒，助您保持健康的使用习惯。

★ 最常用应用： 在主屏幕挂件或通知栏中显示您最常用的应用，方便您快速启动它们。

★ 追踪所有安装： 监控所有应用的安装和卸载行为，让您对手机上的应用变化了如指掌。

★ 应用安装提醒： 在安装新应用后立即弹出通知，并提供每日安装应用概要，方便您管理新安装的应用。

★ 管理应用： 提供一键卸载应用功能，并支持多种排序方式，方便您快速找到并卸载不常用的应用。

重要提示： 由于Android系统的限制，App Usage 只能在您正在使用应用且屏幕处于点亮状态时才能追踪应用使用情况。

► 详细应用使用历史

想知道您在一个应用上投入了多少时间吗？想了解自己每天或平均在特定应用上花费的时间吗？App Usage 可以按您的需求，以各种顺序展示您使用应用的时间统计，让您轻松发现不常用的应用并卸载，还可以监测他人是否使用过您的手机。

► 手机解锁历史

您一天会解锁多少次手机呢？App Usage 可以以条形图或日历视图的形式展示您每天查看手机的次数，让您对自己的手机使用习惯有更清晰的认识。

► 活动历史

想知道您每天打开邮件或短信应用的具体时间吗？App Usage 可以以时间线或日历视图展示您打开应用的时间，帮助您回顾自己的使用轨迹。

► 过度使用提醒

App Usage 可以在您花费过多时间在手机或某个应用上时发出提醒，帮助您保持自律，避免过度使用手机。

► 通知历史

您每天会收到多少通知呢？哪些应用发送的通知最为频繁？App Usage 可以统计您每天收到的通知数量，并显示应用发布通知的时间，让您掌握通知动态。

► 最常用应用

App Usage 可以在桌面挂件或系统通知栏中显示常用应用的列表，让您方便地打开最常用的应用，提高效率。

► 追踪所有安装

想知道是否有未经您允许就被安装或卸载的应用吗？想了解一款应用的更新频率吗？App Usage 可以追踪并按您的排序方式列出所有应用的安装和卸载历史，让您方便地了解每天有多少应用被更新，以及它们的更新频率。

► 应用安装提醒

App Usage 可以提醒您应用新安装的信息及每日安装概要，方便您及时了解并管理新安装的应用。

► 管理应用

想知道有哪些应用已经很久没有使用过了吗？可以通过卸载应用来释放存储空间，但是哪些是系统预装应用呢？App Usage 可以按照应用名称、使用时间、访问次数、更新次数或大小来列出所有应用，让您轻松快速地卸载不再需要的应用。

功能亮点：

★ 手机/应用使用情况、手机活动和查看手机历史记录。

★ 每日使用情况统计、过度使用提醒。

★ 常用应用列表显示。

★ 应用安装历史记录追踪。

★ 应用安装提醒功能。

★ 追踪已卸载应用，方便以后重新安装。

★ 批量卸载应用。

★ Root卸载器，一键卸载应用，需要设备拥有root权限。

★ 给每个应用添加私人笔记。

★ 按应用名称、使用时间、访问次数、更新时间或大小来排序应用。

★ 多列视图显示应用。

★ 批量清除应用缓存或数据。

★ 批量显示用户选中应用的信息。

★ 在Google Play批量显示应用。

★ 通过名称轻松搜索应用。

★ 分享应用列表给朋友。

★ 兼容Android 2~9。
