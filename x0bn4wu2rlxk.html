Scrcpy(手机投屏) v3.3.1 命令行版

Scrcpy通过adb方式连接手机，让你在电脑上也能流畅操控安卓设备，如同华为的投屏功能，支持USB和WiFi连接，此版本为官网原版命令行版，操作简单，适用所有平台。

很多投屏软件都是基于 Scrcpy 二次开发。

Scrcpy 将安卓手机画面投屏到电脑，支持鼠标控制、键盘输入、剪切板共享、文件拖放 (传输及安装 APK)。

Scrcpy 投屏效果清晰流畅，延迟低，可自定义视频码率。自动横屏，操作灵敏，实用性强。

使用教程

https://github.com/Genymobile/scrcpy

准备：USB 数据线，安卓 5.0+，开启开发者选项和 USB 调试。

Scrcpy 使用 adb 驱动通讯，Windows 版已包含 adb。也可手动安装 ADB 命令行工具。

不同手机开启开发者选项方式不同，请自行搜索。

Windows 用户：下载安装，连接数据线，执行 scrcpy.exe 启动。首次连接需授权 USB 调试。

CMD 命令行：

  1. 切换到 Scrcpy 目录：`cd Scrcpy目录`
  2. 运行：`adb devices`
  3. 运行：`scrcpy -s 你的设备ID`

Scrcpy 自动运行并显示安卓设备界面，可像操作手机一样操控。

更多命令设置：[https://github.com/Genymobile/scrcpy](https://github.com/Genymobile/scrcpy)
