MKVToolNix v93.0.0 精简便携版

MKVToolNix 是一套强大的 MKV (Matroska) 文件处理工具集，包含 mkvmerge、mkvinfo、mkvextract、mkvpropedit 和 mmg 等实用程序。它允许你以多种方式操作 Matroska 文件，包括创建、分割、编辑、混流（封装、打包）、分离（分解、解包）、合并、提取或检查 MKV 文件。

该程序不仅支持 MKV 格式，还广泛兼容其他视频格式，例如 AVI、MPEG、MP4、Ogg/OGM、RealVideo、MPEG1/2、h264/AVC、Dirac、VC1，甚至包括 VP9 视频编解码器（支持从 IVF/Matroska/WebM 文件读取，提取到 IVF 文件）。在音频方面，支持 AAC、FLAC、MP2、MP3、(E)AC3、DTS/DTS-HD、Vorbis、RealAudio 等主流格式。此外，它还支持各种字幕格式，如 SRT、PGS/SUP、VobSub、ASS、SSA 等。

MKVToolNix 是一款跨平台的应用程序，可在 Windows（32 位和 64 位版本）、Mac OS X 和 Linux 上运行。完整的软件包包含用于合并和提取流、查看信息以及编辑标题和章节的单独命令行实用程序。Windows 版本提供了一个图形界面，将所有功能集成在一起，方便用户操作。

使用 MKVToolNix，你可以查看、追加和分割曲目、章节和标签，并编辑各种数据。例如，在处理通用轨道选项时，可以设置轨道名称、语言、默认和强制轨道标志、标签和时间码等。此外，你还可以设置宽高比、FPS、延迟、立体影像模式、裁剪、提示、压缩模式以及增强功能的自定义命令行参数。外部文件可以作为附件嵌入到电影中。

在创建和编辑电影章节时，可以定义属性，例如开始和结束时间、标志、段和段版本 UID，以及章节名称、语言和国家。还可以配置标签文件的全局设置、创建 WebM 兼容文件、分割模式、文件链接等。值得注意的是，MKVToolNix 提供了对批处理作业的支持，这意味着你可以配置工作站以无人值守的方式执行耗时任务。你可以检查当前的命令行并将其复制到剪贴板或保存到文件中，以便在将来的项目中使用，管理队列作业，添加命令行选项，从外部文件加载章节，编辑页眉以及将项目另存为 .mka、.mkv 或 .mk3d 格式的文件。

考虑到其广泛的配置参数，MKVToolNix 应该能满足大多数用户学习、创建、编辑和生成 MKV 文件的需求。
