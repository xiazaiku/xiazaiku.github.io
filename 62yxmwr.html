更新日志

功能更新新增 PC 迁移功能：微软正在推出全新 Windows PC 迁移功能，用户可在 Windows 备份应用中看到“欢迎页”和“配对页面”。完整功能推出后，用户在新电脑的初始设置过程中可将旧电脑中的文件和设置直接迁移到新设备。更新 curl 工具：将 Windows 中包含的 curl 工具版本更新至 v8.13.0。功能调整与改进移动运营商配置文件：更新了国家和地区设置资产（COSA）配置文件。默认浏览器设置调整：为欧洲经济区（EEA）推出一系列浏览器改动以遵守欧洲数字市场法案，包括：如果新设定的默认浏览器注册了相关类型的文件和链接格式，系统将自动将这些格式关联到该浏览器。新的默认浏览器将被固定到任务栏和开始菜单，除非用户主动取消。新增一个单独的设置按钮，用于更改.pdf 默认设置。问题修复身份验证平台：修复了一个影响 Entra ID Windows 账户管理器（WAM）插件中的设备注册问题。输入和组合：修复了一个影响未使用的语言包和按需功能（FOD）包完全移除的问题，此前该问题导致了“不必要的存储占用”，并增加了 Windows 更新安装时间。打印和外设：修复了一个影响 USB 连接的多功能打印机的问题，此前具有双协议接口的打印机可能遇到扫描失败/操作系统内置扫描功能无法使用的问题。开始菜单：修复了一个导致跳转列表从开始菜单消失的问题。修复了一个在安装更新后开始菜单无法启动的问题。设置：修复了“设置”>“系统”>“关于”意外显示版本 2009 而不是版本 22H2 的问题。服务：修复了一个使用 ForceAutoLogon 配置和支持管理员解锁后的信息亭设备可能停止响应并显示蓝屏的问题。文件服务器：修复了一个在 SMB 共享资源上确认 Oplock 断开时系统可能挂起的问题。
