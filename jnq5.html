Beyond Compare 是一款强大的文件及文件夹对比工具，旨在帮助用户高效地识别和同步差异。它不仅能快速比较目录结构的不同，还能深入比较每个文件的内容，并以清晰的方式展示比较结果。Beyond Compare 内置了文件浏览器，方便用户在文件、文件夹、压缩包和 FTP 站点之间进行差异比对和数据同步。

Beyond Compare 软件特色：

会话概念的扩展：

  • 所有类型的比较都可以保存为会话，不仅仅是文件夹比较。
  • 插件被替换为内置会话类型，简化了操作流程。
  • 保存的会话可以在文件夹中进行管理，便于查找和组织。
  • 自动保存新的比较到会话，避免了重复设置。

全面支持 Unicode：

  • 不仅支持 Unicode 文件内容，还支持 Unicode 文件名，增强了兼容性。

界面改进：

  • 针对会话视图的选项卡界面，提高了用户体验。
  • 手头的“主视图”用于启动或管理会话，操作更加便捷。
  • 可以保存和恢复多个打开会话的布局，方便用户快速恢复工作环境。
  • 内置“检查升级”功能，方便用户及时更新软件。

文件夹比较功能：

  • 可以选择喜欢的文件夹图标颜色，个性化定制界面。
  • 多个操作可以通过队列执行，提高工作效率。
  • 文件操作默认使用过滤器，过滤不必要的文件。
  • 专用文件夹同步会话，同步更加准确。
  • Touch 和属性命令可以递归执行到子文件夹，简化了操作。
  • 属性命令现在支持 Unix 属性，增强了跨平台兼容性。
  • 新的忽略命令将不匹配文件标记为确定 mismatched file as okay，减少误报。
  • 新的移动到文件夹命令，方便文件管理。
  • 平展文件夹帮助比较文件夹结构的不同，更容易发现差异。
  • 能够替代对齐文件，更加灵活。

文本比较特点：

  • 完整的窗格内编辑器自带动态重新比较，实时显示差异。
  • 高亮语法，便于代码阅读和比较。
  • 自适应装订按钮用于复制节，快速复制代码块。
  • 保存之后无限制的撤销，避免误操作。
  • 基于语法的比较规则，比较更加准确。
  • 两侧比较运用独立的文件类型，更加灵活。
  • 新的忽略 命令标记不匹配文件为确定，减少误报。
  • 多重书签，方便定位和查找。
  • 十六进制显示行细节的选项，方便调试。
  • 三方文本合并，解决代码冲突。
  • 在和另一侧特定文本想比较时，能够将一侧的特定文本视为不重要的，减少干扰。

改进 FTP 支持：

  • FTP 配置文件，每一个会话可选，方便管理多个 FTP 站点。
  • 多项同时连接，提高传输速度。
  • 内置安全 FTP 协议支持（SFTP 和 FTP 基于 SSL），保证数据安全。
