微软 Win11 记事本全面支持 Markdown 格式，普通用户现已可用

7 月 2 日消息，今年 5 月 30 日，微软宣布推出最新的记事本（版本 11.2504.50.0），引入轻量级格式功能，但仅面向 Canary 和 Dev 用户提供。

Windows Latest 发现，Win11 记事本应用已全面上线 Markdown 支持，且并未导致系统性能下降。

非 Windows Insider 计划的普通用户已可通过微软商店更新获取该功能，启动时将弹出确认弹窗（如图所示）。

更新后的记事本具备完整 Markdown 编辑能力，新增功能包括：

点击“H1”按钮可设置标题 / 副标题 / 章节 / 子章节

新增项目符号、编号列表及文本加*号按钮

支持 Ctrl+K 快捷键插入带锚文本的超链接（点击后通过默认浏览器打开）

界面底部增设“格式视图”按钮，可切换 Markdown 风格模式

在 Markdown 模式下，用户可直接使用#号创建标题、* 号强调文本、反引号包裹代码等标准 Markdown 语法，两种视图模式输出结果一致。

实测表明该功能默认开启，但提供两种关闭方案：

点击清除格式按钮暂时禁用（注：保留功能开关）

通过设置 > 关闭“格式”选项永久保持经典模式（同时移除格式工具栏）

本次更新使得 Win11 记事本具备了更强大的文本编辑能力，对于经常需要编写 Markdown 文档的用户来说，无疑是一个好消息。无需安装额外的 Markdown 编辑器，即可在系统自带的记事本中完成文档的编写和预览，极大地提高了工作效率。同时，Markdown 语法的支持也使得记事本能够更好地适应互联网时代的需求，方便用户进行内容创作和分享。

除了基本的 Markdown 语法支持外，新版记事本还提供了一些便捷的工具，例如一键设置标题、加*号、插入链接等，使得 Markdown 的编写更加简单易用。对于不熟悉 Markdown 语法的用户来说，也可以通过这些工具快速上手，体验 Markdown 的优势。此外，新版记事本还提供了两种视图模式，用户可以根据自己的喜好选择 Markdown 风格模式或经典模式，更加灵活地进行文本编辑。

总的来说，Win11 记事本对 Markdown 的全面支持是一次重要的更新，它不仅提升了记事本的功能，也使得 Windows 系统更加贴合用户需求。相信随着 Markdown 的普及，会有越来越多的用户选择使用 Win11 记事本进行文本编辑和创作。
