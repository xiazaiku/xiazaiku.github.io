TagSpaces(文件标签管理器) v6.5.1 绿色版

TagSpaces：打造你的专属文件管理系统

TagSpaces 是一款强大的、无供应商锁定的开源文件管理工具，旨在帮助用户通过标签组织、注释和管理本地文件。它不仅具备高级笔记功能，还融入了待办事项应用程序的特性，让你能够在一个平台上高效处理多种任务。TagSpaces 适用于 Windows、Linux、Mac OS 和 Android 等主流操作系统，并为 Firefox、Edge 和 Chrome 浏览器提供 Web Clipper 扩展，方便用户轻松收集在线内容，并以本地文件形式保存。

核心功能

文件和文件夹管理：TagSpaces 提供直观的用户界面，方便浏览、查看和管理电脑中的文件和文件夹。

灵活的文件标记：TagSpaces 支持两种文件标记方法。你可以选择将标签直接嵌入到文件名中，或者使用 sidecar 文件来存储标签信息，灵活适应你的工作习惯。

强大的搜索功能：利用模糊搜索功能，你可以快速过滤指定位置，查找包含特定标签的文件和文件夹，提高搜索效率。

本地内容浏览器：TagSpaces 可以作为本地照片或电子书库的浏览器，方便你快速浏览和查找所需内容。

内置媒体播放器：TagSpaces 支持播放常见的音频和视频文件类型，并集成了基本的媒体播放器功能，让你无需切换应用即可享受多媒体内容。

完全离线，*安全*可靠：TagSpaces 完全离线运行，无需互联网连接或任何形式的在线注册和订阅，保障你的数据*安全*和隐私。

便捷的笔记功能：你可以创建和编辑纯文本、Markdown 和 HTML 格式的笔记，随时记录想法和灵感。

高效的待办事项管理：利用 HTML 编辑器，你可以在每个 HTML 文件中创建简单的待办事项列表，轻松管理你的任务。

跨平台支持：TagSpaces 可在 Windows、Linux、Mac 和 Android 平台上运行，实现跨设备无缝衔接。

Web Clipper 扩展：通过 Firefox 和 Chrome 浏览器的 Web Clipper 扩展，你可以方便地保存网页和屏幕截图，快速收集网络信息。
