QOwnNotes 开源 Markdown 笔记本 v25.8.6

QOwnNotes 是一款强大的开源记事本，专注于 Markdown 支持和待办事项管理。它完美适配 GNU/Linux, macOS 和 Windows 平台，并能与 Nextcloud Notes 和 ownCloud Notes 无缝集成。

使用 QOwnNotes，您可以随时记录您的灵感，并通过 Nextcloud Notes 或 ownCloud 网络服务在移动设备上轻松编辑和检索。您的笔记以纯文本 Markdown 文件的形式存储，并通过 Nextcloud/ownCloud 的文件同步功能保持同步。当然，您也可以选择 Syncthing 或 Dropbox 等其他同步工具。

如果您追求极致的自由度，喜欢直接访问纯文本文件格式的笔记，例如 Nextcloud/ownCloud notes 应用，那么 QOwnNotes 将是您的理想之选。

QOwnNotes 的主要特性：

  • 多笔记文件夹支持：轻松管理多个笔记存储位置。
  • 现有文件无缝衔接：直接使用已有的文本或 Markdown 文件，无需繁琐的导入过程。
  • 历史版本恢复：从 Nextcloud/ownCloud 服务器恢复旧版本的笔记 (需安装 QOwnNotesAPI)。
  • 已删除笔记恢复：从 Nextcloud/ownCloud 服务器恢复已删除的笔记 (需安装 QOwnNotesAPI)。
  • 本地回收站：安全存放已删除的笔记。
  • 强大的子字符串搜索：快速定位笔记内容，搜索结果高亮显示。
  • 自定义快捷键：根据个人习惯设置应用程序的操作快捷方式。
  • 外部更改监控：实时检测笔记文件的外部更改，并自动重新加载。
  • 差异对比：在对话框中显示当前笔记和外部更改笔记之间的差异。
  • Markdown 高亮和预览：提供清晰的 Markdown 语法高亮和实时预览。
  • 拼写检查：确保您的笔记内容准确无误。
  • 标签支持：使用标签对笔记进行分类和组织。
  • 脚本支持：通过脚本扩展 QOwnNotes 的功能，并可从在线脚本仓库安装脚本。
  • Web Companion 浏览器扩展：方便地从选定的文本和其他功能添加注释 (支持 Chrome 和 Firefox)。
  • 自动重命名：笔记从文本的第一行获取名称，并自动重命名文件。
  • Nextcloud/ownCloud 待办事项管理：管理您的 ownCloud 任务或 Tasks Plus/Calendar Plus 待办事项列表。
  • 笔记加密：内置 AES-256 加密，或使用自定义加密方法，例如 Keybase.io 或 PGP。
  • 暗黑模式：提供舒适的暗黑主题。
  • 高度可定制的界面：所有面板都可以自由放置、浮动或堆叠。
  • 完全可定制的工具栏：根据您的需求自定义工具栏。
  • Freedesktop 主题图标支持：与您的桌面图标和主题完美融合。
  • 分层注释标签和注释子文件夹：更灵活地组织您的笔记。
  • Nextcloud/ownCloud 服务器共享：方便地与他人共享笔记。
  • 便携模式：将 QOwnNotes 放在 USB 随身碟上，随时随地使用。
  • Vim 模式：为 Vim 爱好者提供熟悉的编辑体验。
  • 无干扰模式：全屏模式和打字机模式，专注于写作。
  • Evernote 导入：轻松将 Evernote 笔记导入 QOwnNotes。
  • 多语言支持：支持 60 多种语言。

中文设置方法：

打开菜单 "note -> settings"，在 "interface" 选项卡中找到 "简化字"，设置后重启软件即可。
