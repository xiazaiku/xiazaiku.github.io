微软“套路”深：壁纸应用成推广必应搜索引擎新工具

↓↓下载地址在最下面↓↓

8 月 8 日消息，据 WindowsCentral 报道，微软正采取一些手段诱导 Windows 11 用户切换到 Bing 搜索引擎。

虽然 Bing 和 Edge 在各自领域并未占据主导地位，但 Windows 操作系统却拥有 72.08% 的桌面市场份额。今年早些时候，Mozilla 曾指责微软利用“有害设计”和欺骗手段，为 Edge 浏览器获取不公平优势。

而现在，微软似乎还有新的招数。报道称，微软正在利用 Bing 壁纸应用“推荐不同的浏览器设置”。弹窗通知建议用户将 Bing 设置为 Windows 11 的默认搜索引擎，以获得快速、安全的搜索结果。

注意到，WindowsCentral 编辑尝试复现这一情况，但未能成功，但称 Bing 壁纸应用确实默认勾选了将 Bing 设为主页和默认搜索引擎的选项。用户在安装前应取消勾选，以避免意外更改设置。

Reddit 上的帖子指出，接受微软的“推荐设置”会自动在谷歌 Chrome 中安装 Bing 扩展。如果属实，这可能是微软为增加 Bing 用户而采取的另一种手段，甚至不惜在竞争对手的浏览器中推广自家服务。

目前，这一问题似乎仅限于 Windows Insider Beta 频道的用户，微软可能会在后续更新中修复。


近年来，搜索引擎市场的竞争日益激烈，各大厂商纷纷推出自己的产品，力求在这个信息时代占据一席之地。微软的Bing搜索引擎作为挑战者，一直试图打破谷歌的垄断地位。然而，市场份额的差距依然巨大，促使微软不断探索新的推广策略。

此次事件中，微软被指利用其壁纸应用作为推广Bing的工具，引发了用户的关注和讨论。这种做法的争议之处在于，它可能涉嫌通过非直接的方式诱导用户更改默认搜索引擎设置。尽管微软的目的是推广自己的产品，但用户对于软件擅自更改设置的行为通常持谨慎态度。

从用户体验的角度来看，用户有权自由选择自己喜欢的搜索引擎。任何试图强制或欺骗用户更改设置的行为，都可能损害用户对品牌的信任感。因此，微软需要在推广自身产品的同时，更加注重用户体验和尊重用户的选择。

此外，微软还面临着来自竞争对手的压力。Mozilla 曾公开指责微软利用“有害设计”来推广 Edge 浏览器，这表明行业竞争已经进入白热化阶段。在这种情况下，微软需要更加谨慎地处理推广策略，避免引发更多的争议。

对于用户而言，保持警惕，仔细阅读软件安装过程中的提示信息，是避免被“*路”的有效方法。同时，用户也可以通过反馈渠道向软件厂商表达自己的意见，促使厂商改进产品和服务。

总之，微软利用壁纸应用推广Bing搜索引擎的做法，反映了搜索引擎市场竞争的激烈程度。然而，在追求市场份额的同时，微软也需要兼顾用户体验和尊重用户的选择，才能赢得用户的长期信任。
