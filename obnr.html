DigiKam v8.7.0：开源照片管理利器

digiKam 是一款强大的开源数字照片管理程序，为摄影爱好者和专业人士提供全面的照片管理、编辑和分享解决方案。它跨平台支持 Linux、Windows 和 macOS，让您随时随地轻松管理您的照片库。

轻松导入与整理

使用 digiKam，您可以轻松地从数码相机、SD 卡、U 盘等设备导入照片、RAW 图像文件和视频。自定义导入选项，设定照片处理和整理规则，让您的照片井然有序。

强大的标签与搜索功能

digiKam 提供了强大的标签工具，您可以为照片指定标签、评分和附加标记，并根据这些条件进行筛选，快速找到您需要的照片。此外，其内置的搜索功能支持各种条件，包括标签、标记、评价、日期、地理位置，以及 EXIF、IPTC 和 XMP 元数据，满足您深入搜索的需求。

RAW 图像和视频文件处理

digiKam 通过 LibRaw 程序库处理 RAW 图像文件，该程序库保持活跃更新，支持最新型号的数码相机。同时，它通过 FFmpeg 和 QtAv 程序库管理视频文件，提取元数据并进行播放。

全面的图像编辑工具

digiKam 内置一套功能齐全的图像编辑工具，包括基本的颜色调整、裁剪、锐化，以及颜色曲线调整、全景图拼接等高级功能。Lensfun 程序库的集成更实现了自动镜头畸变校正。

扩展性与自定义

通过插件（DPlugins 工具），您可以扩展 digiKam 的功能，将内容导入/导出到各种在线服务，并获得更多图像编辑和批处理功能。

主要特点

  • 在相册和子相册中组织照片 (支持标签和注释)
  • 支持 EXIF、IPTC、XMP、Makernotes
  • 通过 SQLite 或者 MySQL 提供高级搜索工具
  • 支持对相册进行筛选和排序
  • 支持从 1100 多款数码相机设备导入照片
  • 支持超过 900 种 RAW 图像格式
  • 提供并排比较照片的看片台工具
  • 通过插件扩展额外功能
  • 使用 HTML 共享照片或发布到 WEB 服务
  • 图像编辑器提供用于快速修复照片的各种高级工具
