Omni：美观实用的浏览器标签插件

↓↓下载地址在最下面↓↓

文章来自于公众号首发：https://mp.weixin.qq.com/s/6hsCYjgR-KOxe_AgO6EmCw

最近，一款开源的浏览器插件引起了讨论，在 Github 上面斩获了3.1K 的 stars ！这是款集合了书签、历史记录、浏览标签、三位于一体的页面管理器。

开发者是一位既懂设计又懂开发的外国小姐姐- Alyssa X 她一个人做的，她不仅自己创业开发了12款工具，还是国外知名产品挖掘站点 Product Hunt 的年度制作人，她的作品主要以实用为主。

像是上面这个屏幕录制工具Screenity也是她的作品。

而我们今天的主角-Omni,是一款被网友夸赞能提高生产力的工具，可以在浏览器里对网页进行一系列的快捷操作。不止有：快速切换标签页面、浏览记录、快速滚动、清除浏览历史和Cookie 密码等基本功能，还有50多项联动与其他项目的功能！比如：

发送新邮件开一个新的 Github 仓库开一篇新的 Notion 文档开一个新的 Asana 任务开一个新的 Figma 文件建立一篇新的 Google 表单建立一篇新的 PPT发布一条 Tweet

看上去很是强大，来体验一下，在文末有提供离线包的下载地址，安装之后，按快捷键 Ctrl+Shift+K 呼出 Omni 的主界面就可以使用。

有一点需要注意，有的介绍说的是 Ctrl+K 呼出界面，然而在 Chrome 实测中要按顺序，先按下 Shift 再按住 Ctrl 再按 K 才能呼出界面，很是蛋疼。装了其他插件时，或者浏览器自带快捷键功能，更是有占用的快捷键的情况发生。或着直接在 Edge 浏览器里没有绑定快捷键。

这时候可以在浏览器打开地址栏，输入 chrome://extensions/shortcuts 来跳转到插件的快捷键设置窗口，这个链接在使用谷歌家内核的浏览器上都能打开。

火狐浏览器可以直接点击旁边的按钮，之后设置成无冲突，自己喜欢的快捷键位即可。

下面是一些Omni常用的功能。

/tabs: 搜索你的标签

/bookmarks: 搜索你的书签

/history: 搜索你的浏览器历史

/actions: 搜索所有可用的操作

/remove: 删除书签或关闭标签

使用的时候不用担心记住与输入这么一大串英文，比如想要网页静音，直接输入 /a mu 回车即可，只需要输入命令的第一个字母，非常方便。

熟悉一下后，就可以在浏览器的标签和历史记录里面自由穿梭！有朋友会说，Chrome 里面不是有内置搜索标签页的功能了？Chrome默认的是加载标签，而不是切换，对于新手还是 Omni 这样的工具比较顺手一些。

如果不知道具体的操作，也可以使用鼠标直接浏览 Omni 附带的操作。另外，这个插件会随系统主题自动切换黑夜模式，又是一个小细节。还有比较实用的功能：有时候浏览的网页因为缓存等其他原因出现故障，或者想要清除登录状态，这时候就可以直接用 clear 快速选择清理的项目。

另外在网页上写东西，同时开了多个标签页的时候，这个插件的作用就出来了，可以直接用键盘完成快速切换标签，滚动，固定，等一系列操作，配合浏览器本身自带的快捷键，最大限度的保持写作和思维的连贯性。

还有一些你可以用键盘快捷键，也能用Omni 完成的功能：

滚动到页面底部、顶部

重新加载页面

打印页面

录屏

打开下载页面

打开浏览器设置

返回上一页

去下一页

复制一份当前页面

关闭浏览器

值得一提的是，这个插件和前欧朋浏览器团队关系密切的 Vivaldi 浏览器有些相同之处，下面这是 Vivaldi 的 F2 键快速搜索功能，也能实现类似的操作。

但是好在 Omni 以插件的方式，覆盖了更多浏览器，还没体验过这种方式操作浏览器的朋友，用 Omni 来上手体验是不错的选择。这个插件的评论区也是一片好评。

说到这些好评，也有网友表示这是看到开发者是小姐姐才给的好评。是不是因为小姐姐的姿色不确定，也可能是因为插件比较漂亮。

像这样常年打开一个窗口的场景，又有许多标签使用，还看不到标签名字时，还是有还多发挥的空间的。而联合其他平台的功能，本质上也是一个快捷书签，是打开了其他平台网址来完成的，不能算作插件的功能。建议 Omni 与之前推荐的 浏览器操作自动化工具 搭配食用。
