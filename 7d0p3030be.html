Sigil EPUB电子书编辑器 v2.6.0 官方中文版

ePub（Electronic Publication）是一种开放标准的电子书格式，支持内容自动重排，优化阅读体验。其内部采用 XHTML 或 DTBook（DAISY Consortium 的 XML 标准）来展示文本，并以 ZIP 压缩格式封装内容。ePub 格式可选择包含数位版权管理（DRM）功能。

Sigil 的操作界面与微软 Office 类似，易于上手。它支持导入 TXT、HTML 和 EPUB 等格式，并导出 EPUB 和 SGF（Sigil 原生格式）。软件提供简体中文界面，方便国内用户使用。

Sigil v2.6.0 更新日志

  • 新增菜单加速器 P-&Plugins 和 K-Chec&kpoint (感谢 BeckyEbook)
  • 为 BookBrowser、Preview、CodeView、ClipsWindow 和 TableOfContents Windows 添加使用键盘快捷键或菜单更改键盘焦点的功能
  • 删除未使用的类时保留媒体覆盖活动类
  • 转到链接或样式（或 Ctrl 单击一个类）现在将使用组合器搜索选择器，如果在正常的 css 类中没有找到任何内容
  • 更新到更新的西班牙语汉字词典
  • Linux 留档楼宇大修
  • 更新 3rdparty zlib 到版本 1.3.1
  • 安全和其他 bug 修复
  • 将 3rdparty pcre2 更新到版本 10.43 以获取安全和其他 bug 修复
  • 更新 Qt6.6.2 以进行安全修复、可访问性改进
  • 如果修复保存集且格式不正确，则允许取消保存
  • Windows 用户现在可以将 SIGIL_USE_FREETYPE_FONTENGINE 环境变量设置为更完全支持 woff 和 woff2 字体
  • 错误修复来自 Qt6.6.2 的多个 bug 修复，例如 Windows 上的瞬态子窗口调整大小 bug 等
  • 修复 QuickParser bug 解析属性名称时未正确忽略所有合法空格
  • 使 OPF 解析对替代空格使用也具有鲁棒性
  • 修复在 SelectFiles 和 macOS 图像报告中创建 svg 图像缩略图的问题
  • 通过过滤掉文本标签预渲染中的 desc 和 title 标签来修复 QtSvg 中的 svg 渲染
  • 修复 Python 插件 quickparser.py 以更好地处理 svg 混合大小写标签名称
  • 更好地处理在插件完成之前取消插件
  • 更好地处理指南/地标标题翻译（谢谢 BeckyEbook）
  • 清理 cmake 并构建警告
  • 修复使用 woff/woff2 字体时 Windows 崩溃的问题
