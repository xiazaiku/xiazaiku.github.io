微软Edge游戏助手正式登陆Win11，告别Alt+Tab

微软正式面向所有Windows 11用户开放内置于游戏栏（Win+G）的Microsoft Edge Game Assist，结束了之前的测试阶段。这款工具旨在为PC玩家提供更便捷的游戏体验，无需频繁切换窗口即可访问网络内容。

Edge Game Assist本质上是一款嵌入Win11游戏栏的特制浏览器，自2024年11月推出预览版以来，便致力于解决玩家在游戏过程中查阅攻略、联系好友或收听音乐的需求。微软官方强调，通过Win+G快捷键，玩家可以快速启动Edge Game Assist，避免Alt+Tab切换可能造成的延迟甚至游戏崩溃。

这款浏览器深度融合了Microsoft Edge的核心功能，包括密码管理、收藏夹、历史记录以及扩展程序同步。同时，它还提供专门针对游戏场景的优化服务。

主要功能亮点：

  • 游戏情境感知： 能够自动识别当前运行的游戏，并推送相关的游戏攻略（目前支持超过80款游戏）。
  • 快速检索功能： 通过Ctrl+G快捷键或右键菜单中的“粘贴游戏名称”选项，自动填充当前游戏名称进行网页搜索，节省玩家手动输入的时间。
  • 多任务管理： 支持将浏览器窗口悬浮于游戏画面之上，实现边玩游戏边查看攻略、观看直播等操作，真正做到游戏娱乐两不误。
  • 平台直连： 集成了Discord、Spotify、Twitch等常用平台的快捷入口，方便玩家与好友互动、欣赏音乐和观看游戏直播。同时，Edge Game Assist与主浏览器实现标签页分离管理，避免相互干扰。

微软表示，在测试期间，他们充分听取了用户的反馈意见，并对Edge Game Assist进行了多项改进，例如优化界面视觉设计以突出网页内容、新增键盘快捷键支持、扩展收藏夹与历史记录访问功能等。目前的版本主要针对键鼠操作进行了优化，未来还将进一步完善对游戏手柄的支持。
