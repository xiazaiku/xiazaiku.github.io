OpenHashTab 是一个强大的 Shell 扩展，它允许用户方便地从文件属性中计算和检查文件哈希值。这款工具无缝集成到Windows操作系统中，极大地简化了文件完整性验证的过程。

软件特色

OpenHashTab 提供了卓越的功能，包括：

  • 广泛的算法支持：支持14种不同的可选哈希算法，涵盖了从经典的MD5到现代的SHA-3系列。具体包括md5sum / sha1sum / sha256sum等。
  • 易于使用的校验器：通过直观的界面，用户可以轻松地检查和导出哈希值。只需简单的点击，即可将校验和复制到剪贴板或保存到文件中。
  • 高性能哈希计算：采用优化的计算引擎，确保快速且准确地生成哈希值。
  • 原生Windows外观：完美融合Windows操作系统的界面风格，提供一致的用户体验。
  • 多语言支持：支持包括英语、德语、意大利语、匈牙利语、简体中文、西班牙语、丹麦语和俄语在内的多种语言。
  • 长途支持：

OpenHashTab 与 Windows 8 及更高版本完全兼容。

支持算法

OpenHashTab 支持以下哈希算法：

  • CRC32
  • MD2, MD4, MD5
  • RipeMD160
  • Blake2sp
  • SHA-1
  • SHA-2 (SHA-224, SHA-256, SHA-384, SHA-512)
  • SHA-3 (SHA3-256, SHA3-384, SHA3-512)
