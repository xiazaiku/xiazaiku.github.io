Supermium(第三方Chromium) v132.0.6834.226

探索 Supermium，一款基于 Chromium 的创新 Web 浏览器，专为 Windows XP、2003、Vista、7 和 8.x 等经典操作系统量身定制。Supermium 不仅保留了 Chromium 的强大功能和现代 Web 标准支持，还巧妙地优化了对旧版硬件的兼容性，尤其是那些配备 SSE2 指令集处理器的设备。

核心特性

经典 Aero 风格：Supermium 提供了 Aero Glass 和 Aero Glass 风格的标题栏选项，让您在现代浏览器中重温 Windows 经典界面的魅力。您可以通过 chrome://flags 中的 #force-xp-theme 选项进行设置。

快速安全更新：我们致力于在官方披露后一周内快速修复主要安全漏洞，保障您的网络安全。

强化安全沙箱：Supermium 采用功能强大的安全沙箱，有效隔离恶意代码，提升浏览安全性。

无缝谷歌同步：轻松同步您的书签、历史记录和设置，享受跨设备一致的浏览体验。

DRM 内容支持：在 Windows 7 及更高版本上，Supermium 支持 Widevine CDM，让您可以流畅观看受 DRM 保护的内容。

GDI 字体渲染：通过 chrome://flags 中的 #force-gdi 选项，您可以启用 GDI 字体渲染，获得更清晰的文本显示效果。

持久暗黑模式：在 chrome://flags 中使用 #force-dark-mode 即可为浏览器 UI 元素启用持久暗黑模式，保护您的视力。

自定义标签页选项：Supermium 提供了多种自定义标签页选项，包括梯形标签页、透明标签页和轮廓标签页，让您打造个性化的浏览体验。

摆*来自***的干扰：不再受到***搜索内容的捆绑，享受更纯粹的浏览体验。

SSE2 处理器支持：32 位版本完全支持仅具备 SSE2 指令集的处理器，扩大了兼容性范围。
