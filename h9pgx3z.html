UniGetUI (原 WingetUI) 旨在为 Windows 10 和 Windows 11 上常见的命令行界面 (CLI) 包管理器，如 Winget、Scoop、Chocolatey、Pip、Npm 和 .NET Tool 创建一个图形用户界面 (GUI)。 使用 UniGetUI，您可以轻松下载、安装、更新和卸载受支持包管理器上发布的任何软件。

软件特色

UniGetUI 能够安装、更新和卸载来自 Winget（包括 Microsoft Store）、Scoop、Chocolatey、pip、npm 和 .NET Tool 的软件包。 UniGetUI 甚至能检测您手动安装的应用程序是否可以更新！ 此外，它还可以升级和卸载以前安装的软件包，以及卸载内置的 Windows 应用程序！ UniGetUI 允许您导入和导出选定的软件包，方便您将来轻松安装。 它还支持通过界面管理 Scoop 存储桶，并允许安装旧版本的应用程序。 当有可用更新时，UniGetUI 会显示通知。 通过上下文菜单，您可以管理更新和已安装的软件包。 用户将收到应用程序安装/更新/卸载是否成功完成的通知。 为了防止冲突，您可以对安装进行排队。 深色主题可保护您的视力。 UniGetUI 可以在安装前显示软件包的相关信息，如许可证、SHA256 哈希值、主页等。 如果启用 Winget、Scoop 和 Chocolatey，将有超过 14000 个软件包可用。
