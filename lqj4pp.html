第二块“阿拉伯金显砖”：华硕推出 ROG Astral GeForce RTX 5080 Dhahab CORE OC

6 月 18 日，华硕隆重推出 ROG Astral GeForce RTX 5080 Dhahab CORE OC 显卡，继 ROG Astral GeForce RTX 5090 Dhahab OC 后，再添一款奢华“阿拉伯金显砖”。这款显卡以其独特的设计理念和卓越的性能，再次引领了高端显卡市场的潮流。

RTX 5080 Dhahab CORE OC 延续了 RTX 5090 Dhahab OC 的标志性外观，核心频率与 ROG Astral RTX 5080 OC 保持一致，同为 2970MHz / 2760MHz。然而，Dhahab CORE OC 在外观设计上更具匠心，充分展现了华硕对于艺术与技术的完美融合。

这款 RTX 5080 金显砖在 Astral“夜神”的基础上，大胆采用了金色外观，彰显尊贵与奢华。显卡正面装饰着精美的阿拉伯书法，流露出浓郁的异域风情。背板则巧妙地融入了当地特色的骆驼商队和摩天大楼图案，将传统文化与现代都市完美结合。此外，风扇位点缀着精致的马赛克蔓藤花纹，细节之处尽显奢华。

值得关注的是，华硕此次选择在全球新闻室发布 ROG Astral GeForce RTX 5080 Dhahab CORE OC，并在 ROG 全球官网上线了产品页面，预示着这款显卡将在全球范围内推出，让更多玩家能够体验到其独特的魅力。ROG Astral GeForce RTX 5080 Dhahab CORE OC 不仅仅是一块显卡，更是一件融合了文化与艺术的珍品，将为高端游戏玩家带来前所未有的视觉享受和性能体验。 华硕显卡的又一力作，必将在市场上掀起一股新的金色风暴！
