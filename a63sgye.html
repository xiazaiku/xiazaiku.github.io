Windows 10 终结？KDE 邀您拥抱 Linux 新生


微软即将停止对 Windows 10 的支持，KDE 发起“Windows 10 终结”倡议，呼吁用户转投 Linux 阵营，为您的旧电脑赋予新生。

Windows 10 电脑或将沦为“电子垃圾”？未修复的漏洞、黑客攻击风险、数据泄露危机，以及无法运行新应用、强制硬件升级，KDE 怒斥微软“技术*索”。保留现有硬件，安装 Linux，体验 Plasma 桌面环境，让您的旧机器焕发第二春！

Plasma 桌面环境：熟悉的操作体验，自定义功能，模仿 Windows、macOS 界面，打造专属风格。开源免费，非营利组织开发，多个虚拟桌面，Dolphin 文件管理器，内置服务器和云服务连接工具。

Linux 初体验：KDE 建议，不要期望找到完全相同的 Windows 程序，寻找具有类似功能的 Linux 应用程序。Firefox、LibreOffice 预装，Discover 应用程序，类似应用商店。

告别 Windows 的烦恼：病毒、广告、间谍软件、强制更新、强制账户注册，通通再见！

Plasma 并非唯一选择：GNOME、Xfce、Cinnamon 等桌面环境同样出色。
