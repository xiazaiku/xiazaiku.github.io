软件介绍

UniGetUI (原 WingetUI) 是一款旨在为 Windows 10 和 Windows 11 操作系统中常见的命令行界面 (CLI) 包管理器，如 Winget、Scoop、Chocolatey、Pip、Npm 和 .NET Tool 提供图形用户界面 (GUI) 的工具。通过 UniGetUI，用户可以更轻松地下载、安装、更新和卸载这些包管理器上发布的各类软件。

软件特点

UniGetUI 具有以下显著特点：

  • 集中管理：统一管理 Winget（包括 Microsoft Store）、Scoop、Chocolatey、pip、npm 和 .NET Tool 中的软件包，无需分别使用不同的命令行工具。
  • 自动检测更新：UniGetUI 能够检测到手动安装的应用程序是否有可用更新，并提供便捷的更新方式。
  • 批量操作：支持批量安装、更新和卸载软件包，提高管理效率。
  • 应用卸载：不仅可以卸载通过包管理器安装的软件包，还可以卸载 Windows 内置的应用。
  • 包导入导出：允许用户导入和导出已选择的软件包列表，方便在不同设备或系统上快速部署相同的软件环境。
  • Scoop存储桶管理：提供界面化的 Scoop 存储桶管理功能，方便添加、删除和管理 Scoop 的软件源。
  • 版本控制：支持安装旧版本的应用程序。
  • 通知提醒：在有可用更新时显示通知，确保软件及时更新。
  • 右键菜单集成：通过上下文菜单管理更新和已安装的软件包，操作更便捷。
  • 安装队列：能够对安装任务进行排队，防止安装过程中的冲突。
  • 深色主题：提供深色主题，保护用户的眼睛。
  • 软件包信息展示：在安装前显示软件包的相关信息，如许可证、SHA256 哈希值、主页等，方便用户做出明智的选择。
  • 海量软件包：集成了超过 14000 个可用软件包（启用 Winget、Scoop 和 Chocolatey 后），满足用户的各种需求。

更新日志

v3.3.2 版本更新要点：

  • 品牌重塑：正式更名为 UniGetUI，所有界面元素和相关文件已更新。
  • 设置迁移：安装此版本会自动将设置和其他数据从 WingetUI 文件夹迁移到 UniGetUI 文件夹。
  • .NET 包管理器优化：所有基于 .NET 的包管理器尽可能调用 NuGet API，极大地提高了查找包和加载包详细信息的速度和可靠性，并加载官方包图标。
  • WinGet COM API：实现了 WinGet COM API，以提高性能和可靠性。
  • Microsoft Store 支持：Microsoft Store 软件包（目前仅限 m6）将显示图标和屏幕截图。
  • 图标缓存重建：图标缓存机制已从头开始重建，提供与 Widgets 的更高兼容性。
  • 软件页面改进：优化了发现软件页面，提升搜索体验。
