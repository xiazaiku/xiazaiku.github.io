Task Explorer (进程监控工具) v1.6.5

Task Explorer 是一款强大的任务管理工具，不仅能够监控正在运行的应用程序，还能深入洞察它们正在进行的操作。其用户界面设计注重速度和效率，以最少的交互呈现关于进程的实时数据。Task Explorer 无需多窗口或子窗口，所有相关信息都集中在可访问的面板中。当您选择一个进程时，详细信息会显示在屏幕下半部分，方便您使用箭头键无缝浏览数据。动态数据刷新确保用户能够实时观察变化，从而更清晰地了解系统性能和行为。

软件特色

Task Explorer 提供了一系列高级功能，旨在提供对系统的全面可视性。线程面板显示所选线程的堆栈跟踪，让您即时了解应用程序的当前操作，这对于诊断死锁或性能瓶颈非常有效。内存面板允许用户查看和编辑进程内存，并配备一个高级内存编辑器，支持字符串搜索。在句柄面板中，所有打开的句柄都会显示，包括文件名、当前文件位置和大小等关键细节，让用户清晰地了解程序正在执行的磁盘操作。

套接字面板提供对每个进程所有打开连接或套接字的可见性，并提供额外的数据速率信息。它还可选择显示基于ETW数据的伪UDP连接，从而使用户能够有效地监控网络通信。模块面板列出所有已加载的DLL和内存映射文件，并允许您根据需要卸载或注入DLL。此外，该应用程序还包含各种其他实用面板，如令牌、环境、Windows、GDI和 .NET 面板。

通过双击一个进程，您可以在单独的窗口中打开任务信息面板，从而能够同时检查多个进程。系统监控功能同样强大，它配备了工具栏图表，可显示系统资源（如CPU、句柄、流量和磁盘访问）的实时使用情况。系统信息面板显示所有打开的文件和套接字，并允许用户控制系统服务，包括驱动程序。专用的性能面板提供CPU、内存、磁盘I/O、网络和GPU资源的详细图表，方便监控和优化系统性能。

对于需要更多屏幕空间的用户，系统信息面板可以完全折叠或在单独的窗口中打开，从而最大限度地扩展任务面板的可用区域。
