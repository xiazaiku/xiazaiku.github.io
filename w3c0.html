CopyQ 剪贴板增强工具 v11.0.0 官方便携版

CopyQ 是一款开源的 QT 编写的剪切板管理工具。它不仅支持基本的文本复制粘贴，还具备强大的搜索和编辑历史功能，以及诸多高级特性。用户可以利用 CopyQ 管理和存储文本、HTML、图像等多种格式的数据，极大地提升工作效率。

该工具兼容 Linux、Windows 和 macOS 10.9+ 操作系统。用户可以快速浏览和过滤剪贴板历史记录，对项目进行排序、创建、编辑、删除、复制/粘贴以及拖放等操作，并能为每个项目添加注释或标签，方便管理。

CopyQ 提供了全系统的快捷键支持，用户可以自定义命令，通过快捷方式或托盘菜单快速粘贴项目。其外观也具有高度的可定制性，满足不同用户的个性化需求。

更高级的功能包括命令行界面和脚本编写，方便用户进行自动化操作。此外，CopyQ 还可以忽略从特定窗口复制的剪贴板内容，或包含特定文本的剪贴板内容，保护用户*私*。

CopyQ 还内置了类似 Vim 的简单编辑器和快捷键，方便用户进行文本编辑。此次更新主要包括：

  • 选项卡项目限制增加至 100,000。
  • 新增适用于 M1/arm64 架构的 macOS 版本。
  • 新增适用于 arm/arm64 架构的 Debian/Raspbian 版本。
  • 允许使用覆盖项目激活 paste()。
  • 允许重写脚本函数来处理项目添加、删除、更改和加载等事件。
  • 允许通过覆盖 onItemsRemoved() 脚本函数取消删除项目。
  • 允许覆盖剪贴板监视器进程 currentClipboardOwner() 使用的当前剪贴板所有者。
  • 即使搜索输入框处于焦点状态，也允许使用 Ctrl+C 复制项目。
  • Linux：添加构建选项以禁用 X11 支持。
  • Linux：添加构建选项以禁用自动启动。
  • Windows 二进制文件（64 位）现在默认安装到“ProgramFiles”。
  • Windows 和 macOS 版本现在基于较新的 Qt 6。
  • 避免从密码管理器访问剪贴板。
  • 选项卡中的大数据项现在存储在单独的位置，除非选项卡的同步或加密插件处于活动状态。
  • 命令对话框现在显示内置命令的高级属性。
  • 全局快捷方式现在在菜单中也可见。
  • 避免将所有图像格式粘贴为新项目。
  • 显示命令现在也适用于托盘菜单项。
  • Linux：仅当项目内容与选择的开始或结束匹配时，最后存储的文本项目才会从任何新的鼠标选择中更新。
  • 更新 Font-Awesome 6.5.1 的图标字体。
  • 修复插件和命令的拖放顺序。
  • 修复 gpg 2.1 及更高版本的密钥管理问题。
  • 修复添加标签时使用同步插件创建重复项目的问题。
  • 使用同步插件修复冲突的注释和文本。
  • 修复运行脚本后删除的全局对象。
  • Wayland：修复了将图像复制到另一个应用程序实例的问题。
