UltraSearch Professional 是一款高效的文件和文件夹搜索工具，专为 Windows 平台设计。它利用 NTFS 驱动器的 MFT (主文件表) 直接访问文件信息，无需建立索引，从而实现闪电般的搜索速度。只需几秒钟，即可在本地 NTFS 驱动器上找到所需的文件和文件夹，极大地提升您的工作效率。

核心特性：

  • 极速搜索：直接读取 NTFS 分区 MFT，无需索引，实现毫秒级搜索速度。
  • 实时显示：输入文件名或掩码（例如 *.exe）时，立即显示搜索结果。
  • 正则表达式支持：提供高级搜索功能，可使用正则表达式进行模糊匹配。
  • 详细信息显示：显示文件大小、修改日期、访问日期、创建日期等详细信息。
  • 右键菜单集成：UltraSearch 窗口中提供 Windows 资源管理器上下文菜单，操作便捷。
  • 排除过滤：允许创建文件夹、文件和文件类型的排除规则，精确控制搜索范围。
  • 结果排序与导出：搜索结果可按各种标准排序，并导出为 TXT、RTF、HTML、CSV 和 Excel 格式。
  • 搜索历史：保存最近使用的搜索模式，方便快速调用。
  • 自动填充：提供搜索掩码并自动填充手动输入的模板。
  • 资源管理器集成：可从 Windows 资源管理器上下文菜单启动，在选定目录中搜索。
