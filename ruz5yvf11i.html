Heroic Games Launcher是一款适用于 Linux、Windows 和 macOS 的开源游戏启动器，旨在统一您的游戏体验。该启动器目前支持 Amazon、GOG 和 Epic Games 平台，让您可以在一个地方集中管理您的游戏库。通过Heroic，您可以方便地查看游戏详情、购买心仪的游戏、高效整理游戏收藏，并且能够根据个人喜好自定义启动器的主题，打造个性化的游戏中心。

软件特色

Heroic Games Launcher 具备以下核心功能，提升您的游戏体验：

  • 多平台账号支持：无缝登录 Epic Games、GOG 或 Amazon 帐户。
  • 游戏管理：轻松安装、卸载、更新、修复和移动游戏，掌控您的游戏文件。
  • 游戏导入：快速导入已安装的游戏，无需重新下载。
  • 在线畅玩：直接通过 Heroic 启动器在线畅玩 Epic 游戏。
  • 兼容性增强：利用 Wine 或 Proton 技术，畅玩 Windows 平台游戏。
  • Crossover 支持：集成 Crossover，提供更多游戏兼容性选项。
  • 自定义 Wine/Proton：下载和管理自定义 Wine 和 Proton 版本，优化游戏性能。
  • 内置商店：直接从 Heroic 访问游戏商店，发现新游戏。
  • ProtonDB 集成：在 ProtonDB 上搜索游戏，获取兼容性信息，了解游戏在不同环境下的运行情况。
  • 兼容性显示：清晰显示 ProtonDB 和 Steam Deck 兼容信息，方便您选择适合设备的游戏。
  • 云同步：将已安装的游戏列表同步到云端，随时随地访问您的游戏库。
  • 云存档同步：支持云端同步保存游戏进度，告别存档丢失的烦恼。
  • 自定义主题：根据个人喜好自定义启动器主题，打造个性化界面。
  • 下载队列：有序管理下载任务，提高下载效率。
  • 扩展性：添加其它游戏和应用程序，将 Heroic 打造成您的专属应用中心。
