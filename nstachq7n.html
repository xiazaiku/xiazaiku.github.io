Microsoft PowerToys v0.92.1 官方版

Microsoft PowerToys v0.92.1：高级用户 Windows 效率利器

Microsoft PowerToys 是一套由微软官方出品的系统增强实用工具集，专为高级用户设计，旨在优化和定制 Windows 体验，从而显著提高工作效率。这款工具的灵感来源于 Windows 95 时代的 PowerToys 项目，它为 Windows 10 和 Windows 11 用户提供了一种途径，能够深度定制系统功能，针对个人工作流程进行优化，发掘系统潜力，摆*传统Windows使用的束缚，让系统更*性化，打造更高效的个人工作空间。

核心功能抢先看

  • 增强型文件资源管理器：通过插件扩展文件资源管理器的功能，提供更强大的文件预览和管理能力。
  • PowerToys Run（聚合搜索）：快速启动应用程序、搜索文件或执行命令，效率倍增。
  • Image Resizer（图片大小调整）：批量调整图片大小，方便快捷。
  • Keyboard Manager（键盘管理器）：自定义键盘映射，打造个性化输入体验。
  • PowerRename（批量重命名）：轻松批量重命名文件，告别繁琐操作。
  • Shortcut Guide（快捷键指南）：显示当前窗口的快捷键，快速掌握操作技巧。

系统要求

推荐 Windows 10 v1903（内部版本 18362）或更高版本，最低支持 Windows 10 v1803（内部版本 17134）。老版本可能需要安装 .NET Core 3.1。

更新日志亮点

  • 修复了文件资源管理器插件中预览不适用于每个用户安装的 Bug。
  • 修复了命令面板中应用程序扩展缺少所有 Win32 应用程序的回归问题。
  • 修复了命令面板每次加载应用时重置应用状态的回归问题。
