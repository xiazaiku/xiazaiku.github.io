微软 Win11 系统今年夏末将推出新的意外重启界面，适配 24H2 版本

Win11 24H2：微软革新意外重启体验


微软宣布，将在今年夏末的 Windows 11 24H2 版本中推出全新的意外重启界面，旨在减少中断，提高用户生产力。

在《Windows 韧性计划：为未来企业构建韧性》博客中，微软强调了企业保持生产力和减少中断的重要性。针对意外重启可能造成的延误，微软简化了相关体验，提升系统的整体韧性。

此次更新的核心在于：

  • 更快的机器恢复（QMR）： 针对无法成功重启的 PC，引入了一种新的恢复机制。QMR 旨在自动化修复过程，无需 IT 团队进行复杂的手动干预，从而快速恢复用户的工作状态。尤其是在大规模停机时，QMR 可以通过 Windows 恢复环境 (Windows RE) 部署有针对性的修复措施。

  • 优化的崩溃转储收集： Windows 11 24H2 改进了崩溃转储收集功能，将大多数用户遇到的意外重启停机时间缩短至约两秒。

  • 简化的用户界面 (UI)： 全新的 UI 设计提高了可读性，更符合 Windows 11 的设计语言。同时，在需要时保留了必要的屏幕技术信息。

QMR 将与重新设计的意外重启界面一同推出，并支持所有 Windows 11 版本 24H2 设备。在 Windows 11 家庭版中，QMR 默认启用。对于企业系统，IT 管理员可以完全掌控该功能，并在运行 Windows 11 Pro 和 Enterprise 的设备上启用它。微软计划在今年晚些时候发布更多用于自定义 QMR 的功能。

这意味着，无论是个人用户还是企业用户，都将体验到更快速、更流畅的系统恢复过程，从而减少意外重启带来的负面影响。
