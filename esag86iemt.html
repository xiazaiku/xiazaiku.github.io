VisualCppRedist v92 运行库合集

Visual C++ 运行库 优化合集

Visual C++ 运行库合集包，精简优化版，集成常用运行库组件，体积小巧，是系统装机必备。

精简版软件常因缺少依赖库而报错，安装本运行库合集可避免此类问题，确保软件正常运行。

本合集包含多个版本的Visual C++ Redistributable组件，覆盖常用运行库，兼容性更强。

包含组件

  • Microsoft Visual Basic/C++ Runtime (x86/x64)
  • Microsoft C Runtime Library (2002: 7.0.9975.0)
  • Microsoft C Runtime Library (2003: 7.10.6119.0)
  • Microsoft Visual C++ Redistributables (x86/x64)
    • Microsoft Visual C++ 2005 Redistributable - 8.0.61187
    • Microsoft Visual C++ 2008 Redistributable - 9.0.30729
    • Microsoft Visual C++ 2010 Redistributable - 10.0.40219
    • Microsoft Visual C++ 2012 Redistributable - 11.0.61135
    • Microsoft Visual C++ 2013 Redistributable - 12.0.40664
    • Microsoft Visual C++ 2015 Redistributable - 14.10.25008
    • Microsoft Visual C++ 2005-2022 Redistributable - 14.44.35211.0
  • Microsoft Visual Studio 2010 Tools for Office Runtime (x86/x64)

安装参数

VisualCppRedist_AIO_x86_x64.exe 支持以下参数：

  • /ai - 自动静默安装所有
  • /aiR - 自动静默卸载所有
  • /y - 正常安装所有，显示安装界面
  • /aiM - 自选安装，每个版本有确认对话框
  • /aiU - 自动卸载，每个版本有确认对话框
  • /aiA - 自动静默安装所有，隐藏ARP项
  • /ai5 - 仅安装Microsoft Visual C++ 2005
  • /ai8 - 仅安装Microsoft Visual C++ 2008
  • /aiX - 仅安装Microsoft Visual C++ 2010
  • /ai2 - 仅安装Microsoft Visual C++ 2012
  • /ai3 - 仅安装Microsoft Visual C++ 2013
  • /ai9 - 仅安装Microsoft Visual C++ 2019
  • /aiT - 仅安装Microsoft Visual Studio 2010 Tools
  • /aiE - 仅安装额外的Microsoft Visual Basic/C++
  • /aiV - 仅安装Microsoft C Runtime Library基础库
  • /aiH - 隐藏已安装项
  • /aiP - 手动隐藏/显示已安装项
  • /aiC - 完全自动被动模式，安装所有，但UCRT KB3118401除外
  • /aiD - 调试模式，创建VCpp_debug.log，不安装/卸载
  • /gm2 - 禁用所有其它开关的提取解压对话框
