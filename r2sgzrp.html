eSearch：一款全能型截图OCR神器，挑战传统截图软件

文章来自于公众号首发：https://mp.weixin.qq.com/s/6zqcx0CTpbMIUSlpHb8oUQ

在电脑上的截图软件，你是否也觉得它们功能单一、体验*肋？今天，我要介绍一款颠覆你认知的全能型工具——eSearch，它集“截屏+OCR+搜索+翻译+贴图+以图搜图+录屏”于一身，挑战传统截图软件的底线！

eSearch，不仅仅是一款截图工具，它更像是一位贴心的助手，在细节之处彰显匠心：取色器、图片编辑、多种应用打开方式、优雅的模糊效果、自动删除换行、以及自定义搜索，每一个功能都经过精心设计，只为提升你的使用体验。

多平台支持，无缝切换

eSearch基于 Electron 构建，完美支持 Windows、Linux 和 MacOS 三大主流平台，无论你使用哪种系统，都能轻松上手，无缝切换。

在官网下载时，你还可以通过预览窗口提前体验软件界面和功能设置，这种贴心的设计，让人倍感惊喜。

首次打开软件，会提示下载 OCR 字符识别的离线包（约 91M），强烈建议下载，离线使用，无需担心网络问题和次数限制。

下载完成后，按照提示安装依赖，即可正常使用。

简洁界面，强大功能

使用快捷键 ALT+C 即可调出 eSearch 的主界面，其简洁的设计风格与 Windows 11 完美融合。

截图时，除了手动调整边框，你还可以通过手动输入或四则运算来精确控制截图大小，配合放大镜功能，选择目标更加轻松。

钉图功能，高效便捷

截图后，你可以选择保存或将图片“钉”在桌面上，方便随时查看。eSearch 还支持无极透明度调节，让你自由调整图片的显示效果。

内置录屏，轻松记录

eSearch 还内置了录屏功能，支持 mp4、gif、mkv、webm 等多种格式，弥补了某些截图工具的不足。如果你喜欢 QQ 截图，又需要录屏功能，不妨在等待其完善的同时，先用 eSearch 解解馋。

配置 FFmpeg，释放录屏潜力

使用录屏功能前，需要先下载并配置 FFmpeg，这是编解码与媒体处理的必备工具。下载解压后，将 FFmpeg 的 Bin 文件夹路径添加到软件设置中，注意按照使用文档的提示，在路径后加上 \ffmpeg.exe 文件名。

配置完成后，你就可以像使用常规录屏工具一样，录制声音、设定输出格式和码率，甚至直接输出为 GIF 动图。

图片编辑，随心所欲

eSearch 提供了强大的图片编辑功能，其中最吸引人的莫过于其优雅的打码功能——毛玻璃效果，比传统的像素块更加美观。

此外，你还可以进行字符识别和以图搜图等操作。

离线 OCR 字符识别，精准高效

eSearch 的离线 OCR 字符识别功能表现出色，如果你对精度有更高要求，也可以配置自己申请的 API 接口。

识别翻译，一键直达

识别出的文字结果可以直接跳转到百度或谷歌翻译，或者直接使用搜索引擎进行快速搜索。

以图搜图，溯源追踪

想要查找图片出处或了解图中信息？只需截图后点击识别，即可快速跳转到识图结果，省去手动上传的繁琐步骤。

识别二维码，便捷提取

eSearch 还具备识别二维码的功能，截图后点击识别按钮，即可轻松解析出二维码中的内容。

长截图，滚动截屏

备受期待的长截图（滚动截屏）功能也已加入 eSearch，虽然目前还在测试阶段，拼接过程可能需要一些时间，但相信未来会更加完善。保存长图片时，软件会自动全屏展示截取的内容，属于正常现象，手动点击保存即可。

调用其他软件，灵活扩展

如果需要进行更深入的编辑，你可以直接使用其他图片编辑软件打开截取的图片。
