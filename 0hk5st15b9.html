中文设置指南

您可以通过以下步骤设置中文界面：

  1. 进入游戏 Settings 菜单。
  2. 选择 Game 选项卡。
  3. 在 Language 下拉菜单中选择“中文”。
  4. 点击 OK 按钮。
  5. 重启游戏即可生效。

若以上方法无效，请手动修改配置文件：

  1. 打开“我的文档”文件夹，找到 \Paradox Interactive\Victoria 3 目录。
  2. 使用记事本打开 pdx_settings.json 文件。
  3. 将文件内容替换为以下代码并保存：{"Graphics":{"display_mode":"fullscreen","display_index":"0","vsync":true,"fullscreen_resolution":"2560x1440"},"System":{"language":"l_simp_chinese"}}
  4. 右键点击 pdx_settings.json 文件，选择“属性”，勾选“只读”选项，点击“应用”和“确定”。
