QOwnNotes：开源Markdown笔记本，极致自由的纯文本笔记体验

QOwnNotes是一款强大的开源记事本，专为追求极致自由和灵活性的用户设计。它完美支持 Markdown 语法，并提供针对GNU / Linux, Mac OS X和Windows的全平台支持。通过与Nextcloud Notes 和 ownCloud Notes 的无缝集成，您可以随时随地记录和编辑您的想法。

核心特性：

  • 纯文本存储：所有笔记均以纯文本 Markdown 文件存储，保证数据的持久性和可移植性。
  • 云同步：完美兼容Nextcloud / ownCloud的文件同步功能，您也可以选择 Syncthing 或 Dropbox 等其他同步方案。
  • 多文件夹支持：轻松管理多个笔记文件夹，分类整理您的知识体系。
  • 无缝导入：直接使用现有的文本或 Markdown 文件，无需繁琐的导入过程。
  • 版本控制：借助Nextcloud / ownCloud服务器，轻松还原旧版本或已删除的笔记。
  • 强大的搜索功能：支持子字符串搜索，搜索结果高亮显示，快速定位所需信息。
  • 自定义快捷键：根据您的习惯，自定义键盘快捷方式，提高工作效率。
  • 外部修改监控：实时监控笔记文件的外部更改，确保数据同步。
  • Markdown 高亮与预览：提供Markdown语法高亮和实时预览，优化您的写作体验。
  • 拼写检查：内置拼写检查功能，减少您的笔误。
  • 标签支持：使用标签组织笔记，方便查找和管理。
  • 脚本支持：通过脚本扩展 QOwnNotes 的功能，满足您的个性化需求。
  • Web Companion 扩展：使用 QOwnNotes Web Companion 浏览器扩展，快速添加来自网页的笔记。
  • 自动重命名：笔记名称自动从第一行文本获取，并自动重命名文件 (可关闭)。
  • 待办事项管理：管理 Nextcloud / ownCloud 待办事项列表 (ownCloud tasks 或 Tasks Plus / Calendar Plus)。
  • 笔记加密：内置 AES-256 加密，或使用 Keybase.io 或 PGP 等自定义加密方法，保护您的隐私。
  • 暗黑模式：提供暗黑模式主题，保护您的眼睛。
  • 高度可定制：所有面板均可自由停靠、浮动或堆叠，打造您专属的工作环境。
  • Freedesktop 图标主题支持：与您的桌面图标和深色主题完美融合。
  • 分层标签和子文件夹：支持分层标签和子文件夹，更灵活地组织您的笔记。
  • Nextcloud / ownCloud 共享：在 Nextcloud / ownCloud 服务器上共享笔记，方便协作。
  • 便携模式：将 QOwnNotes 放入 USB 随身碟，随时随地使用。
  • Vim 模式：为 Vim 爱好者提供 Vim 模式。
  • 无干扰模式：全屏模式和打字机模式，助您专注写作。
  • Evernote 导入：轻松从 Evernote 导入笔记。
  • 多语言支持：支持 60 多种语言。

中文设置方法

打开菜单 Note -> Settings，在Interface选项卡中找到 "简化字"，设置后重启即可。
