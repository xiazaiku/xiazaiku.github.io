Ungoogled-Chromium，一款源自 GitHub 的开源浏览器项目，是 Chromium 的精简优化版本。它致力于提供开源、免费、无广告的浏览体验，通过移除所有 Google 的网络服务功能，包括烦人的自动更新，来保障用户的隐私安全和提升浏览器性能。目前，Ungoogled-Chromium 完美支持 Windows、MacOS、Linux 等主流操作系统。

在功能体验上，除了缺少 Google 服务集成外，Ungoogled-Chromium 与 Chrome 浏览器几乎没有差别。为了实现更彻底的隐私保护，该浏览器在关闭时会自动清除所有网站的 Cookie 信息。

如何安装拓展

默认情况下，此浏览器不支持直接从在线商店安装扩展程序。但您可以通过以下步骤解决：

1. 首先，在地址栏中输入：chrome://flags/#extension-mime-request-handling，找到 "Extension MIME Request Handling" 选项，并将其设置为 "Always prompt for install"。如果已经设置为该选项，请跳过此步骤。

2. 接下来，安装此扩展（如果无法直接打开，请尝试使用*理）：Chromium Web Store

3. 安装完成后，您就可以在线安装其他扩展程序了。
