Lunacy，由知名图标社区 icons8 精心打造，是一款专为 Windows 平台设计的 Sketch 文件查看与编辑利器，尤其适用于 HTML/CSS 开发环境。它能让你在 Windows 上无缝打开并读取原本为 macOS 专属的 .sketch 文件，解决设计师在跨平台协作时的燃眉之急。

尽管当前版本功能相对精简，主要集中在基础编辑和对齐操作上，但 Lunacy 的发展蓝图已经绘就。在未来的版本迭代中，我们有望看到阴影和效果、网络布局、高级文本处理、渐变渲染以及演示模式等更多强大功能的加入，使其成为一款全能的 UI 设计辅助工具。

近年来，Sketch 已成为 UI 设计师的新宠，但其 macOS 独占的特性给 Windows 用户带来了不便。为了打破这一壁垒，icons8 的工程师倾力开发了 Lunacy。它不仅能让你在 Windows 系统上完整预览 .sketch 文件，更提供了一系列便捷的编辑功能，极大地提升了工作效率。告别繁琐的格式转换，直接在 Windows 上畅享 Sketch 文件的魅力。

  • 基础编辑功能：提供基本的编辑、对齐等操作，满足日常设计需求。
