常见问题

1、打开 Firefox 后显示原版界面？请确保文件路径不包含中文或空格，例如 "Program Files" 这样的路径。尝试更换解压路径，并以管理员身份运行。

2、选项出现 "Windows Script Host 错误：无效字符"？在 "火箭图标 -> 脚本文件夹" 中，右键点击 "userChrome.js"，选择 "属性"，然后在打开方式中选择 "记事本"。或者，点击 "火箭图标 -> 系统工具 -> 关联记事本"，使用 Notepad2 替代默认记事本。

3、如何升级更新？

小幅度更新：点击 "火箭图标 -> 更新修复 -> 自动更新" 一键更新，并删除 "Cache" 缓存文件夹。

大幅度更新：

1. 下载更新的压缩包后，登录 Firefox 帐号，书签会自动同步。建议手动更新扩展和用户脚本，避免同步出错。

2. 转移旧版书签和历史记录：将旧版配置文件夹下的 "\Profiles\V10\places.sqlite" 覆盖到新版同名文件。

3. 书签图标空白：将旧版配置文件夹下的 "\Profiles\V10\favicons.sqlite" 覆盖到新版同名文件。

4. 转移旧版搜索引擎列表：将旧版配置文件夹下的 "\Profiles\V10\search.json.mozlz4" 覆盖到新版同名文件。

5. 如有自定义修改，请备份修改内容，然后重新添加到新版本。
