微软 Win11 正式版支持一键更改默认浏览器

Win11 告别繁琐：一键切换默认浏览器时代来临


微软 Win11 系统浏览器设置的痛点终于迎来突破！用户长期以来诟病的默认浏览器更改难题，在最新版本中得到显著改善。告别过去繁琐的操作，Win11 正式版现在支持一键切换默认浏览器，极大地提升了用户体验。

挑战与改进：

此前，Win11 更改默认浏览器的流程复杂且易出错，用户需手动修改多个文件关联，引发诸多不满。微软甚至被指责未经用户许可重置浏览器设置，引发争议。面对开发者和用户的强烈呼声，微软最终做出改变，在最新更新中恢复了一键更改功能。

KB5011563：关键更新

通过安装 Windows 11 可选累积更新 KB5011563，用户可以体验到这一便捷功能。该更新不仅修复了大量 Bug，还将设置应用程序的改进从 Dev 预览版带到了稳定版。现在，用户只需进入默认应用程序设置，选择心仪的浏览器，点击按钮即可将其设置为默认浏览器。

操作指南：

以下是在安装 Windows 11 Build 22000.593 或更高版本后更改默认浏览器的步骤：

  1. 确保已安装您希望设置为默认的浏览器。
  2. 打开“设置”应用程序（Win + I），导航至“应用程序”>“默认应用程序”。
  3. 在应用程序列表中找到您想要设置的浏览器并打开它。
  4. 点击“设置默认”按钮。

Edge 浏览器的特殊地位

尽管一键切换功能已恢复，微软仍然在某些方面保留了 Edge 浏览器的优先权。例如，Edge 仍然是打开 PDF、SHTML、SVG、FTP 和其他网络相关文件的默认应用程序。此外，用户可能仍然需要第三方工具来指定 Windows 小组件中的链接在其他浏览器中打开。尽管如此，一键切换功能的回归无疑是 Win11 在用户体验方面迈出的重要一步。
