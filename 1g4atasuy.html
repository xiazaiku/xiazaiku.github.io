QOwnNotes 开源 Markdown 笔记本 v25.7.8

QOwnNotes 是一款开源的记事本应用，它强大之处在于对 Markdown 的完美支持以及集成的待办事项列表管理功能。无论你使用的是 GNU / Linux、Mac OS X 还是 Windows，QOwnNotes 都能流畅运行，并且可以与 Nextcloud Notes 和 ownCloud Notes 无缝协作。

随时随地记录，高效同步

你可以使用 QOwnNotes 随时记录你的想法，并且通过 Nextcloud Notes 或 ownCloud 的 Android 应用，在移动设备上轻松编辑或搜索这些笔记。QOwnNotes 将笔记存储为纯文本 Markdown 文件，并利用 Nextcloud / ownCloud 的文件同步功能进行同步，确保你的数据安全可靠。

灵活的同步选择

除了 Nextcloud / ownCloud，你还可以使用 Syncthing 或 Dropbox 等其他同步工具，充分满足你的个性化需求。

纯文本至上，自由掌控

如果你追求对笔记的完全掌控，喜欢像 Nextcloud / ownCloud notes Web 应用程序那样直接访问纯文本文件，那么 QOwnNotes 将是你的理想选择。它为你提供最大的自由度，让你可以随心所欲地管理你的知识库。

QOwnNotes 的核心特性

  • 支持多个笔记文件夹，灵活管理你的知识体系。
  • 直接使用现有的文本或 Markdown 文件，无需繁琐的导入过程。
  • 可从 Nextcloud / ownCloud 服务器恢复旧版本或已删除的笔记（需安装 QOwnNotesAPI）。
  • 强大的子字符串搜索功能，并高亮显示搜索结果。
  • 可自定义的键盘快捷键，提升操作效率。
  • 实时监测笔记文件的外部更改，自动重新加载。
  • Markdown 语法高亮和预览功能，所见即所得。
  • 支持拼写检查、标签和脚本，扩展性强。
  • 集成 QOwnNotes Web Companion 浏览器扩展，方便从网页添加笔记。
  • 自动从笔记首行获取名称，并自动重命名文件（可关闭）。
  • 管理 Nextcloud / ownCloud 待办事项列表（兼容 ownCloud tasks 或 Tasks Plus / Calendar Plus）。
  • 强大的加密功能，内置 AES-256，并支持自定义加密方法，如 Keybase.io 或 PGP。
  • 暗黑模式主题，保护你的眼睛。
  • 完全可停靠的面板，自定义你的工作区。
  • 支持 freedesktop 主题图标，完美融入你的桌面环境。
  • 支持分层注释标签和注释子文件夹。
  • 可在 Nextcloud / ownCloud 服务器上共享笔记。
  • 便携模式，方便在 USB 闪存盘上携带使用。
  • Vim 模式和无干扰模式，专注写作。
  • 支持 Evernote 导入。
  • 支持 60 多种语言，感谢社区的贡献。

QOwnNotes 中文设置方法

打开菜单 Note -> Settings，在第五项 Interface 中找到“简化字”，设置后重启即可。
