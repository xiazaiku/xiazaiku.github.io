Joplin：你的私有化开源笔记利器

Joplin 是一款强大的开源笔记和待办事项应用，横跨 Linux、Windows、macOS、Android 和 iOS 全平台。它以用户隐私和数据安全为核心，提供灵活的笔记管理和同步方案，是追求自主掌控和数据安全用户的理想选择。

核心优势：

  • 完全开源： 自由使用、修改和分发，社区支持强大。
  • 私有化部署： 数据可存储在本地或私有服务器，无需担心数据泄露。
  • 多端同步： 通过 NextCloud、Dropbox、OneDrive 或 WebDAV 等服务，实现数据在不同设备间的无缝同步。
  • Markdown 支持： 采用 Markdown 格式，方便编辑和排版，支持数学公式和复选框。
  • 端到端加密： 提供端到端加密选项，确保笔记内容的安全性和隐私性，即使同步到第三方服务也无需担心*密泄露。
  • 数据迁移便捷： 提供多种导入和导出功能，轻松迁移笔记数据，告别厂商*定。
  • 免费使用： 作为开源软件，Joplin 没有任何隐藏费用。
  • Web Clipper： 快速保存网页内容，方便资料收集和整理。
  • 插件支持： 通过插件扩展功能，满足个性化需求。

Joplin 提供了三种视图模式：Markdown 代码、Markdown 预览和并排模式，方便用户根据自身习惯进行编辑。此外，Joplin 还支持添加附件（带图像预览），并在外部 Markdown 编辑器中编辑，每次保存时自动更新。Web Clipper 插件可轻松保存网页内容。
