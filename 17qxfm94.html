更新日志

新功能任务栏缩放：用户可以在设置应用的“个性化 > 任务栏 > 行为”部分中，启用“显示更小的任务栏”选项，确保始终使用小图标，而不再仅当任务栏充满应用后图标才缩小。PC 迁移应用：微软正在推出全新 Windows PC 迁移功能，用户可在 Windows 备份应用中看到“欢迎页”和“配对页面”。完整功能推出后，用户将能够在新电脑的初始设置过程中将旧电脑中的文件和设置直接迁移到新设备上。浏览器设置调整：为欧洲经济区（EEA）推出一系列浏览器改动以遵守欧洲数字市场法案。如果新设定的默认浏览器注册了相关类型的文件和链接格式，系统将自动将这些格式关联到该浏览器；新的默认浏览器将被固定到任务栏和开始菜单，除非用户主动取消；新增一个单独的设置按钮，用于更改.pdf 默认设置。任务栏与托盘改进：重构了快捷设置中的“辅助功能”菜单，按视觉、听觉、动作和行动能力分组，同时整合了语音访问功能；调整了任务栏图标下方指示器（小条形）的宽度，使其更明显易见。Windows 分享改进：用户通过 Windows 分享窗口分享链接或网页内容时，系统将显示预览图像；在编辑并分享图像时，Windows 分享窗口将提供三档图像压缩质量可选（高、中、低），不再使用 0–100 的滑动刻度。文件资源管理器改进：改进了压缩文件的解压性能，重点优化了 7z/rar 格式大型压缩包解压能力，性能提升大约 15%，解压含大量小文件的压缩包时表现最为明显。讲述人改进：新增“屏幕遮罩”功能，可将屏幕变黑，仅保留语音朗读功能，从而提升隐私与专注度，适用于用户在公共环境中处理敏感信息；现在在界面中提供更丰富的功能引导，便于新用户学习和现有用户探索更新内容。语音访问改进：用户现在可以使用中文通过语音命令操作系统；用户现在可以在语音访问功能中添加自定义词汇。设置改进：对于由 IT 管理员维护的电脑，设置主页新增企业专属卡片，允许 IT 管理员自由配置“推荐设置”、“辅助功能偏好”等内容。如果用户同时登录了工作/学校账号与微软账号，系统还会显示额外的账号信息卡片；在“设置 > 时间和语言 > 语言和区域”中，现在会显示设备初次设置时所选的国家或地区。窗口改进：修复从全屏游戏使用 ALT + Tab 切出后，其他窗口（如 Windows Terminal）可能无响应的问题；修复了某些设备从睡眠状态恢复后，窗口大小和位置可能出现变化的问题；修复启用窗口贴靠功能时，拖动窗口可能导致 explorer.exe 异常退出的问题；修复了在较旧的 Windows Server（如 2019）版本上运行 SMB 共享中的脚本时，执行速度异常缓慢的问题。图形改进：优化了显示体验，包括“减少在切换显示配置时的屏幕闪烁”等特性；修复了部分显示器画面可能意外呈现绿色的问题；修复了当“用户账户控制（UAC）”设为“始终通知”时，点击“设置 > 系统 > 显示”下的颜色校准按钮并取消时，设置应用可能出现未响应的问题；调整“设置 > 辅助功能 > 颜色滤镜”下“强度”和“颜色增强”滑块的位置，使顶部颜色预览在调整时始终可见。输入法改进：修复了使用触控键盘输入日文后切换为英文键盘再切回日文输入时可能无法正常输入的问题。打印改进：修复了打印时的线条可能比预期更粗的问题。MSFTEdit.dll 修复：修复了当系统语言设为阿拉伯语或希伯来语时，某些应用（如便利贴、dxdiag）可能崩溃的问题。
