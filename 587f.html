Table Habit 功能详解

Table Habit 采用评分系统来追踪您的微习惯。通过每天记录习惯的完成情况，应用程序会根据您的表现给予评分。这种量化的反馈能够帮助您更好地了解自己的进步，并激励您坚持下去。此外，Table Habit 支持正向和负向习惯，让您可以全面管理您的生活方式。您可以设置每天锻炼、阅读等正向习惯，也可以记录避免*烟*酒、熬夜等负向习惯。应用程序的易用性是 Table Habit 的一大亮点。简洁的界面让您可以快速记录习惯的完成情况，并查看您的进度。您还可以为不同的习惯设置不同的颜色，以便更好地区分和管理。Table Habit 还支持将习惯数据导出为 JSON 格式，方便您备份和迁移数据。此外，应用程序还适配了 Android 12 及 Material3 Dynamic Color，能够根据您的系统主题自动调整颜色，带来更加美观的用户界面体验。最后，Table Habit 没有任何广告，让您可以专注于您的习惯养成，不受任何干扰。
