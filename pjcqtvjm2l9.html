微软 Win10 支持即将结束：调查显示 12% 的用户不清楚自己用的系统版本

Win10 终结在即：用户升级意愿调查与潜在风险

微软 Windows 10 的生命周期进入倒计时，计划于 2025 年 10 月 14 日停止支持。迁移至 Windows 11 迫在眉睫，然而，TechRadar Pro 最新调查揭示，高达 53% 的 Windows 用户依然坚守 Windows 10 阵地，这无疑增加了他们在支持停止后面临网络安全威胁的风险。

调查选取了 1027 名用户，结果显示仅有 43% 的受访者完成了从 Windows 10 到 Windows 11 的升级。究其原因，除了硬件兼容性问题，用户对自身系统版本的认知不足也是一大阻碍。

55% 的受访者表示明确知晓自己使用的 Windows 版本，但仍有 23% 的用户“比较确定”，10% 的用户“只能猜测”，更有 12% 的用户完全不清楚。考虑到 TechRadar Pro 的受众偏向技术爱好者，普通用户群体中对系统版本一无所知的比例可能更高。

进一步分析，61% 的受访者家中拥有两台或更多设备，其中 53% 的人表示所有设备均满足 Windows 11 的升级要求。然而，28% 的受访者承认存在硬件短板，14% 的人不了解 Windows 11 的系统要求，4% 的人甚至不清楚自己设备的配置情况。这种对系统要求的模糊认知无疑延缓了他们的升级步伐。

Windows 11 的最低内存要求为 4GB，但超过半数（53%）的受访者对此表示茫然。在声称知道内存容量的 40% 的人中，约四分之一给出了明显不合理的答案，例如 1TB 内存的说法，这暴露出用户对自身设备配置的认知存在严重偏差。

尽管 Windows 11 的市场份额正在逐步提升，但大量用户仍未完成升级。微软正积极推动用户更新，通过在 Windows 11 中集成人工智能功能，吸引用户升级。然而，用户认知不足、硬件限制以及对新系统的不信任感，都成为升级路上的绊脚石。距离 Windows 10 停止支持的时间越来越近，用户应尽快了解自身设备是否符合升级要求，并采取行动，确保系统安全。
