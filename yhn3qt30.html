API开放：打造高效数据通道

为了更好地服务开发者和用户，正式开放API接口，旨在提供便捷、高效的数据获取方式，助力APP开发和个性化信息订阅。本API接口每30分钟更新一次，确保数据的时效性。

最新文章API

通过此接口，您可以获取最新的文章列表，默认返回30条数据。该接口适用于构建新闻聚合应用、个性化推荐系统等。

API地址：ghapi?type=query&n=new

返回数据格式为JSON，包含以下字段：

  • code: 状态码，200表示成功。
  • msg: 状态信息，例如"获取成功"。
  • data: 数据主体，包含以下内容：
    • count: 返回数据条数。
    • list: 文章数据数组，每个元素包含：
      • title: 文章标题。
      • time: 文章发布时间。
      • url: 文章链接地址。
    • last_time: 最后更新时间戳。
  • note: API接口版本信息。

示例：

{    "code": 200,    "msg": "获取成功",    "data": {        "count": 30,        "list": [            {                "title": "文章标题",                "time": "2024-10-27",                "url": "article/123"            }        ],        "last_time": 1730000000,        "note": "API接口1.0"    }}
