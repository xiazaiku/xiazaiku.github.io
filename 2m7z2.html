NotepadNext v0.12：文本编辑新体验

探索 NotepadNext，一个使用 Qt5 工具包构建的 C++ 文本编辑器，界面与 Notepad++ 惊人地相似。尽管仍处于积极开发阶段（当前版本 0.12），NotepadNext 旨在提供类似于 Notepad++ 的功能和用户体验，为用户提供一个稳定且功能丰富的选择。

主要特性

  • 工作区管理： 将文件夹作为工作区，方便组织和管理项目文件。
  • 多语言支持： 支持 C++、YAML、XML、LaTeX、PHP、Python、R、Rust 等多种编程语言，满足不同开发需求。
  • UI 自定义： 灵活调整 UI 设置，启用或禁用部分界面元素，打造个性化编辑环境。
  • 宏录制与执行： 录制、运行和保存宏，简化重复性任务，提高工作效率。
  • 强大的查找与替换： 在文件中快速查找和替换文本，支持正则表达式。
  • 便捷的行操作： 提供多种行操作功能，如复制、剪切、删除、移动等。
  • 文档信息查看： 编辑检查器提供文档详细信息，包括字符数、行数、文件大小等。
  • 缩进指南： 显示缩进指南，帮助保持代码整洁和可读性。
  • 撤销/重做： 无限制的撤销/重做操作，确保编辑过程安全可靠。
  • 放大/缩小： 自由调整文本显示大小，适应不同视觉需求。
  • Lua 控制台： 集成 Lua 控制台，方便进行脚本编写和测试。
  • EOL 操作： 支持多种 EOL (End of Line) 操作，适应不同平台和文件格式。
  • 大小写转换： 快速转换文本大小写，提高编辑效率。

更新日志（v0.12）

  • 允许自定义工具栏。
  • 允许指定多个自定义快捷方式。
  • 突出显示搜索结果。
  • 更新 Notepad 在 Linux 桌面上的外观。
  • 标记令牌出现与特定颜色。
  • 更新 Lexilla 和 Scintilla 组件。
  • 修复潜在的易受攻击的克隆 lua 函数。
